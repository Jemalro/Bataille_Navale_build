Bienvenue à tous ! Pour profiter du jeu, il vous suffit de suivre cette procédure simple :

Téléchargement : Rendez-vous dans la section Releases de ce dépôt et téléchargez la dernière version du fichier Bataille-Navale-Launcher-Setup-X.X.X.exe.

Installation : Lancez l'exécutable.

Note : Il est possible que Windows Defender affiche une alerte au lancement. Vous pouvez poursuivre l'installation en toute confiance. Pour plus de transparence, vous pouvez consulter les mentions légales et les CGU sur notre site officiel : (lien à metttre ici).

Téléchargement du jeu : Une fois le launcher installé, le jeu se téléchargera automatiquement.

Mises à jour : Vous n'avez plus rien à gérer ! À chaque lancement, le launcher vérifie et installe automatiquement la dernière version disponible pour que vous soyez toujours à jour.

Une fois ces étapes terminées, votre jeu est prêt à être lancé !
